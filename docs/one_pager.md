# ProjectOS — One-Pager

## Core idea

Modern AI-assisted work has a structural failure: context evaporates, tasks do not reliably become time, and AI-generated outputs are hard to trust.

ProjectOS is a modular work-context platform that helps knowledge workers capture important work signals, maintain durable project brains, convert actions into protected calendar time, and verify AI-generated updates.

**Tagline:** Capture the signal. Build the project brain. Protect the time. Verify the work.

## Platform loop

```text
LOOK     Capture signals from work surfaces
THINK    Maintain durable project brains
PLAN     Convert actions into calendar commitments
DO       Execute inside protected work blocks
VERIFY   Check AI-generated updates and writes
```

## Core modules

- **ProjectOS Core** — durable project brains
- **ProjectOS Capture** — structured signal capture
- **ProjectOS Time** — task-to-calendar planning
- **AgentOS Controls** — verification and trust layer

## Design principles

- Human approval at write boundaries
- Local-first and inspectable by default
- AI for interpretation; code for invariants
- Adapter-based, not app-specific
- Scope/private/public boundaries are first-class
- Start in review mode; earn automation gradually
